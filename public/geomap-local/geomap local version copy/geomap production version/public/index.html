In this updated version, a LatLngBounds object is created to define the southwest and northeast corners of the desired restricted area. The setOptions method is used to set the restriction property of the Map object to the defined LatLngBounds. Adjust the coordinates in the LatLng objects as needed to define the desired restricted area. With the map restriction in place, users will be limited to viewing and interacting with the map within the specified area, preventing excessive zooming out and scrolling across beyond the defined bounds.
